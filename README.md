# JURA - Kanban Project Tool

This is an example project of using Apollo GraphQL for state management along with Jotai.

## Setup Instructions

### Console 1 (API)
```bash
cd jura-api
nvm use
npm install
npm run start:dev
```

### Console 2 (Client)
```bash
cd jura-client
nvm use
npm install
npm run dev
```
